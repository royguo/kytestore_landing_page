概览

KyteStore 面向湖仓系统和 AI 训推场景，提供 S3 兼容对象协议、POSIX 文件语义和远程对象存储透明加速能力。

核心目标是在统一的数据底座上同时服务对象访问和文件访问，降低多套存储系统并行维护的复杂度。

统一架构

  • ChunkServer 负责统一的数据分片、复制、本地缓存和冷热分离。
  • S3 Subsystem 提供对象语义、S3 API Endpoint 和对象元数据能力。
  • FS Subsystem 提供命名空间、文件元数据和 POSIX/FUSE 接入能力。

数据路径

KyteStore 支持 local only 和 mixed 两类存储模式。本地 SSD 可用于热数据加速，远端 S3/COS/OSS 可作为容量层和回源数据层。

协议接入

当前重点覆盖 S3 兼容对象协议和 POSIX/FUSE 文件语义。架构上可以扩展更多接入协议，按业务需要适配到统一 ChunkServer 数据底座。

部署模式

最小形态支持单机模拟集群部署，便于 PoC 和开发验证；生产形态可按 FE、数据面和控制面拆分部署，逐步增加资源投入。
